# Redirect traffic going to any IP
On REMnux:
```
accept-all-ips start
accept-all-ips stop
```

Using iptables:
```
iptables -t nat -A PREROUTING -i eth0 -j REDIRECT
iptables -t nat -D PREROUTING -i eth0 -j REDIRECT
```
