## Linux

### kismet

```text
kismet -c <mon_interface>
```

### airodump-ng

```text
airodump-ng <mon_interface>

# press [SPACE] to pause scanning
# press [TAB] to select wireless network
# press [S] for sorting
# press [M] for color coding a network and it's clients
```
